يوفر هذا المستند دليلاً خطوة بخطوة لفحص واستعادة وتهيئة محرك الأقراص المحمول (Flash Drive) على نظام لينكس.

---

## الخطوة 1: التحقق من معلومات القرص

اعرض قائمة بجميع محركات الأقراص المتصلة لتحديد الجهاز الصحيح:

```bash
sudo fdisk -l
```

- يقوم هذا الأمر بسرد جميع أقسام القرص وتفاصيل مثل حجم القرص، وحجم القطاع، وتخطيط القسم.
- **المخرجات** ستساعدك في تأكيد السعة الفعلية لمحرك الأقراص المحمول (على سبيل المثال، `/dev/sdb`).

---

## الخطوة 2: الفحص بحثاً عن أخطاء

قم بتثبيت واستخدام `smartctl` لإجراء فحص للصحة:

1. تثبيت حزمة `smartmontools`:

   ```bash
   sudo pacman -S smartmontools
   ```

2. الحصول على معلومات مفصلة عن الجهاز:

   ```bash
   sudo smartctl -i /dev/sdb
   ```

3. إجراء اختبار صحة غير تدميري (إذا كان مدعوماً):

   ```bash
   sudo smartctl -t long /dev/sdb
   ```

- **ملاحظة**: إذا فشلت الاختبارات مع ظهور أخطاء مثل "unsupported SCSI opcode"، فقد لا يدعم الجهاز ميزات SMART.

---

## الخطوة 3: مسح القرص

لمسح جميع البيانات، بما في ذلك جدول الأقسام:

```bash
sudo dd if=/dev/zero of=/dev/sdb bs=1M count=10
```

- **شرح الأمر**:
  - `if=/dev/zero`: ملف الإدخال هو `/dev/zero` (ينتج بايتات صفرية).
  - `of=/dev/sdb`: ملف الإخراج هو محرك الأقراص المحمول.
  - `bs=1M`: حجم الكتلة هو 1 ميجابايت.
  - `count=10`: كتابة 10 كتل (10 ميجابايت).

---

## الخطوة 4: إنشاء جدول أقسام جديد

قم بتشغيل `fdisk` لإنشاء تخطيط قسم جديد:

```bash
sudo fdisk /dev/sdb
```

- **الأوامر داخل `fdisk`**:
  1. إنشاء جدول أقسام جديد (MBR):

     ```bash
     o
     ```

  2. إنشاء قسم أساسي (primary partition):

     ```bash
     n
     ```

  3. حفظ التغييرات على القرص:

     ```bash
     w
     ```

التحقق من جدول الأقسام:

```bash
sudo fdisk -l /dev/sdb
```

---

## الخطوة 5: تهيئة القسم

قم بتهيئة القسم الذي تم إنشاؤه حديثاً بنظام الملفات الذي تريده:

- للأنظمة التي تعمل بنظام لينكس فقط (ext4):

  ```bash
  sudo mkfs.ext4 /dev/sdb1
  ```

- للتوافق بين أنظمة التشغيل المختلفة (FAT32):

  ```bash
  sudo mkfs.vfat -F 32 /dev/sdb1
  ```

---

## الخطوة 6: تحميل القسم (Mount)

1. إنشاء نقطة تحميل:

   ```bash
   sudo mkdir /mnt/flashdrive
   ```

2. تحميل القسم:

   ```bash
   sudo mount /dev/sdb1 /mnt/flashdrive
   ```

3. التحقق من التحميل:

   ```bash
   df -h
   ```

---

## اختياري: اختبار محرك الأقراص للكشف عن المنتجات المقلدة

إذا كان محرك الأقراص يظهر سعة غير صحيحة، استخدم `f3` للتحقق من حجمه الحقيقي:

1. تثبيت `f3`:

   ```bash
   sudo pacman -S f3
   ```

2. اختبار المحرك:

   ```bash
   sudo f3probe --destructive --time-ops /dev/sdb
   ```

---

## الخاتمة

باتباع هذه الخطوات، يمكنك استعادة وإعداد محرك أقراص محمول، والتأكد من استعادة سعته ووظيفته بالكامل. لمزيد من الأتمتة، يمكنك إضافة القسم إلى ملف `/etc/fstab` لضمان تحميله تلقائياً عند الإقلاع.
