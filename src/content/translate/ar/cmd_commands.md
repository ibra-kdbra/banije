## ASSOC

:::note[الغرض]
يُستخدم الأمر assoc لعرض أو تعديل ارتباطات امتدادات الملفات.
:::

```sh
assoc
```

سيعرض هذا الأمر ارتباطات الملفات الحالية.

## CHKDSK (فحص القرص)

:::tip[الغرض]
فحص القرص بحثاً عن أخطاء، وإصلاحها في حال تم طلب ذلك.
:::

```sh
chkdsk C:
```

سيفحص هذا الأمر القرص C: بحثاً عن أي أخطاء.

## CIPHER

:::warning[الغرض]
تشفير أو فك تشفير الملفات/المجلدات على أقسام NTFS.
:::

```sh
cipher /E
```

سيقوم هذا الأمر بتشفير المجلد المحدد.

## CLS

:::note[الغرض]
مسح الشاشة من جميع الأوامر التي تم إدخالها مسبقاً والنصوص الأخرى.
:::

```sh
cls
```

سيقوم هذا الأمر بمسح شاشة موجه الأوامر.

## DISKPART

:::tip[الغرض]
إدارة أقسام القرص الصلب.
:::

```sh
diskpart
```

سيبدأ هذا الأمر مترجم أوامر DiskPart.

## DRIVERQUERY

:::warning[الغرض]
عرض قائمة ببرامج تشغيل الأجهزة المثبتة وخصائصها.
:::

```sh
driverquery
```

سيعرض هذا الأمر قائمة بجميع برامج تشغيل الأجهزة المثبتة.

## GPUPDATE

:::note[الغرض]
تحديث إعدادات نهج المجموعة (Group Policy).
:::

```sh
gpupdate /force
```

سيفرض هذا الأمر تحديثاً فورياً لإعدادات نهج المجموعة.

## IPCONFIG

:::tip[الغرض]
عرض إعدادات IP لجميع واجهات الشبكة.
:::

```sh
ipconfig /all
```

سيعرض هذا الأمر بيانات إعدادات IP التفصيلية لجميع واجهات الشبكة.

## NETSTAT

:::warning[الغرض]
عرض اتصالات TCP النشطة وإحصائيات الشبكة المختلفة.
:::

```sh
netstat -a
```

سيعرض هذا الأمر جميع اتصالات TCP النشطة ومنافذ TCP و UDP التي يستمع إليها الكمبيوتر.

## NSLOOKUP

:::note[الغرض]
الاستعلام عن DNS للحصول على تعيين اسم النطاق أو عنوان IP.
:::

```sh
nslookup <www.example.com>
```

سيعيد هذا الأمر عنوان IP الخاص بـ <www.example.com>.

## PATHPING

:::tip[الغرض]
توفير معلومات حول زمن انتقال الشبكة وفقدان البيانات في القفزات الوسيطة.
:::

```sh
pathping <www.example.com>
```

سيوفر هذا الأمر معلومات حول زمن الانتقال والفقد لكل قفزة شبكية للوصول إلى <www.example.com>.

## PING

:::note[الغرض]
اختبار قدرة الكمبيوتر المصدر على الوصول إلى كمبيوتر وجهة محدد.
:::

```sh
ping <www.example.com>
```

سيرسل هذا الأمر طلب شبكة إلى <www.example.com> وينتظر الرد.

## POWERCFG

:::warning[الغرض]
التحكم في إعدادات الطاقة وتكويناتها.
:::

```sh
powercfg /hibernate on
```

سيعمل هذا الأمر على تفعيل وضع الإسبات.

## SFC (مدقق ملفات النظام)

:::note[الغرض]
فحص وإصلاح ملفات النظام المهمة.
:::

```sh
sfc /scannow
```

سيقوم هذا الأمر بفحص جميع ملفات النظام المحمية على الفور.

## SHUTDOWN

:::tip[الغرض]
إيقاف تشغيل الكمبيوتر، أو إعادة تشغيله، أو تسجيل الخروج منه أو من كمبيوتر بعيد.
:::

```sh
shutdown /s /t 0
```

سيقوم هذا الأمر بإيقاف تشغيل الكمبيوتر فوراً.

## SYSTEMINFO

:::note[الغرض]
عرض معلومات تكوين مفصلة حول الكمبيوتر ونظام التشغيل الخاص به.
:::

```sh
systeminfo
```

سيعرض هذا الأمر معلومات تفصيلية عن النظام.

## TASKLIST

:::tip[الغرض]
عرض قائمة بالمهام والخدمات والعمليات قيد التشغيل.
:::

```sh
tasklist
```

سيعرض هذا الأمر قائمة بالمهام التي تعمل حالياً.

## TRACERT

:::warning[الغرض]
إظهار المسار والقفزات التي تتخذها الحزمة للوصول إلى وجهتها.
:::

```sh
tracert <www.example.com>
```

سيعرض هذا الأمر المسار الذي تسلكه الحزمة للوصول إلى <www.example.com>.
