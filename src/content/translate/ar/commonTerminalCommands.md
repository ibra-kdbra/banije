## نظرة عامة

يقدم هذا الدليل قائمة شاملة بأوامر الطرفية (Terminal) الأساسية، منظمة حسب فئات مثل التنقل، وإدارة الملفات، وغيرها. وهو مثالي للمبتدئين والمستخدمين المتقدمين على حد سواء.

## الأوامر الأساسية والتنقل

:::tip[اختصارات لوحة المفاتيح]
يمكن لاختصارات لوحة المفاتيح توفير الوقت عند التنقل واستخدام الطرفية.
:::

- `Up Arrow`: لعرض آخر أمر قمت بتنفيذه
- `Down Arrow`: لعرض الأمر التالي
- `Tab`: للإكمال التلقائي للأوامر
- `Ctrl + L`: لمسح الشاشة
- `Ctrl + C`: لإلغاء أمر ما
- `Ctrl + R`: للبحث عن أمر سابق
- `Ctrl + D`: للخروج من الطرفية

## أمر الدليل (Manual)

:::note[استخدام أمر `man`]
يعرض أمر `man` أدلة الاستخدام للأوامر على أنظمة Linux و MacOS. استخدم `--help` للحصول على وظيفة مشابهة على Git Bash.
:::

```bash
man ls
```

على Git Bash أو Windows:

```bash
ls --help
```

## أمر `whoami`

:::important[تحديد المستخدم الحالي]
يعرض أمر `whoami` اسم المستخدم الذي سجل الدخول حالياً.
:::

```bash
whoami
```

## أمر `date`

:::tip[الحصول على التاريخ والوقت الحاليين]
يعرض أمر `date` التاريخ والوقت الحاليين.
:::

```bash
date
```

## التنقل في نظام الملفات

:::note[أوامر التنقل الأساسية]
يعد التنقل في نظام الملفات أمراً جوهرياً لاستخدام الطرفية.
:::

| الأمر | الوصف |
| ----------------------------------- | --------------------------------------------------------------------------------- |
| pwd | عرض مسار دليل العمل الحالي |
| ls | عرض محتويات الدليل |
| ls -a | عرض المحتويات بما في ذلك الملفات المخفية (الملفات التي تبدأ بنقطة) |
| ls -l | عرض المحتويات مع مزيد من التفاصيل بما في ذلك الأذونات (عرض تفصيلي) |
| ls -r | عرض المحتويات بترتيب عكسي |
| cd | الانتقال إلى الدليل الرئيسي (Home) |
| cd [dirname] | الانتقال إلى دليل محدد |
| cd ~ | الانتقال إلى الدليل الرئيسي |
| cd .. | الانتقال إلى الدليل الأب (المستوى الأعلى) |
| cd - | الانتقال إلى الدليل السابق |
| find [dirtosearch] -name [filename] | العثور على موقع برنامج ما |

يمكنك دمج الخيارات، مثل `ls -la` لعرض الملفات التفصيلية والمخفية معاً.

## فتح مجلد أو ملف

:::note[فتح الأدلة، الملفات، أو الروابط]
تختلف الأوامر باختلاف أنظمة التشغيل لفتح الملفات أو المجلدات أو روابط الـ URL.
:::

- Mac: `open [dirname]`
- Windows: `start [dirname]`
- Linux: `xdg-open [dirname]`

```bash
open https://example.com
```

## تعديل الملفات والأدلة

:::important[أوامر إدارة الملفات والأدلة]
تعرّف على كيفية إنشاء، حذف، نقل، وإعادة تسمية الملفات والأدلة.
:::

| الأمر | الوصف |
| --------------------------- | --------------------------------------------------- |
| mkdir [dirname] | إنشاء دليل |
| touch [filename] | إنشاء ملف |
| rm [filename] | حذف ملف |
| rm -i [filename] | حذف ملف مع طلب التأكيد |
| rm -r [dirname] | حذف دليل |
| rm -rf [dirname] | حذف دليل قسرياً |
| rm ./\* | حذف كل شيء في المجلد الحالي |
| cp [filename] [dirname] | نسخ ملف |
| mv [filename] [dirname] | نقل ملف |
| mv [filename] [filename] | إعادة تسمية ملف |

إنشاء أدلة متداخلة:

```bash
mkdir -p ./home/{a,b}/{x,y,z}
```

تشغيل أوامر متعددة:

```bash
cd test2 && mkdir test3
```

## قوس الزاوية الأيمن `>`

:::tip[إعادة توجيه المخرجات]
إعادة توجيه مخرجات الأمر إلى ملف.
:::

```bash
> [filename]
```

للخروج استخدم `Ctrl+D`.

## أمر `cat`

:::note[دمج الملفات]
يقوم `cat` بعرض أو إنشاء الملفات ودمجها.
:::

```bash
cat [filename]
cat > [filename]
cat >> [filename]
```

عرض أرقام الأسطر:

```bash
cat -n [filename]
```

## أمر `less`

:::tip[عرض محتويات الملف]
التنقل عبر ملف باستخدام `less`.
:::

```bash
less [filename]
```

للخروج اضغط `q`.

## أمر `echo`

:::important[عرض النص أو الكتابة في الملفات]
عرض النص في الطرفية أو كتابته في الملفات.
:::

```bash
echo "Hello World"
echo "Hello World" > [filename]
```

## أمر `nano`

:::note[تعديل الملفات النصية]
`nano` هو محرر نصوص سهل الاستخدام.
:::

```bash
nano [filename]
```

للخروج اضغط `Ctrl+X`. للحفظ اضغط `Y`.

## أمري `head` و `tail`

:::tip[عرض أجزاء من الملف]
عرض بداية الملف (`head`) أو نهايته (`tail`).
:::

```bash
head -n 5 [filename]
tail -n 5 [filename]
```

## أمر `grep`

:::note[البحث في محتوى الملف]
البحث عن أنماط نصية داخل الملفات.
:::

```bash
grep [searchterm] [filename]
```

## أمر `find`

:::important[تحديد موقع الملفات أو الأدلة]
البحث عن الملفات بالاسم، النمط، أو الخصائص.
:::

```bash
find [dirname] -name [filename]
```

إنشاء ملفات تجريبية:

```bash
touch file-{001..100}.txt
find . -empty
```

حذف الملفات:

```bash
find . -name "file-*" -delete
```

## الربط (Piping)

:::tip[إعادة توجيه المخرجات]
تمرير مخرجات أمر ما إلى أمر آخر.
:::

```bash
find . -name "file-0*" > output.txt
cat output.txt
```

## إنشاء رابط رمزي (Symlink)

:::note[إنشاء اختصارات]
إنشاء روابط رمزية للملفات.
:::

```bash
ln -s [filename] [symlinkname]
```

للحذف استخدم:

```bash
rm [symlinkname]
```

## ضغط الملفات

:::important[إدارة الأرشيف]
إنشاء واستخراج ملفات الأرشيف (tarballs) باستخدام `tar`.
:::

| الأمر | الوصف |
| ----------------------------------- | -------------------------- |
| tar czvf [dirname].tar.gz [dirname] | إنشاء أرشيف |
| tar tzvf [dirname] | عرض محتويات الأرشيف |
| tar xzvf [dirname].tar.gz | استخراج الأرشيف |

## أمر `history`

:::note[سجل الأوامر]
عرض وتنفيذ الأوامر السابقة.
:::

```bash
history
!100
```

تنفيذ الأمر رقم 100 من السجل.
