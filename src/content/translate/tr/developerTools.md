## 🛠️ Web Geliştirme Araçları

Bir web uygulaması mühendisi olarak sürekli olarak becerilerimi ve üretkenliğimi artıracak araçlar arıyorum. Aşağıda, 2024 ve 2025'teki son gelişmeleri içeren temel araçların güncellenmiş bir listesi bulunmaktadır.

---

## Mevcut Araçlar

### 🛠️ Wappalyzer

:::important[Amaç]
Web sitelerinde kullanılan teknolojileri tanımlar.
:::

**Neden Kullanmalı**: Web sunucusunu, CMS'yi, çerçeveleri ve daha fazlasını tanımlamaya yardımcı olur. Diğer web sitelerinden bilgi edinmek ve kendi projelerinize uygun teknolojileri keşfetmek için harikadır.

---

### 🎨 Stylus

:::important[Amaç]
Web sitesinin görünümünü özelleştirin.
:::

**Neden Kullanmalı**: Herhangi bir web sitesindeki CSS stillerini özelleştirin. Tasarımlarla denemeler yapmak ve yazı tipi boyutu veya renk şemaları gibi düzen sorunlarını düzeltmek için kullanışlıdır.

---

### 🧑‍💻 JSON Görünümü

:::important[Amaç]
JSON verilerini biçimlendirin ve doğrulayın.
:::

**Neden Kullanmalı**: API'lerden JSON verilerinin okunmasına ve anlaşılmasına yardımcı olur. Verileri düzgün bir şekilde biçimlendirir ve söz dizimi veya mantıksal hatalarda hata ayıklamaya yardımcı olur.

---

### 📜 HTML Doğrulayıcı

:::important[Amaç]
Hatalar için HTML kodunu kontrol edin.
:::

**Neden Kullanmalı**: HTML kodunun geçerli olmasını ve standartlara uygun olmasını sağlar. Kod kalitesini ve web sitesi performansını artırmaya yardımcı olur.

---

### 📏 Ölç

:::important[Amaç]
Bir web sayfasındaki öğeleri ölçün.
:::

**Neden Kullanmalı**: Bir web sitesindeki görsellerin, metinlerin, düğmelerin ve diğer öğelerin hassas ölçümüne olanak tanır. Düzeni ve tasarımı optimize etmek için kullanışlıdır.

---

### 🌐 REST İstemcisi

:::important[Amaç]
API'leri test edin ve hata ayıklayın.
:::

**Neden Kullanmalı**: Herhangi bir API'ye HTTP istekleri göndermenize, yanıtı görüntülemenize ve durum kodlarını ve verileri analiz etmenize olanak tanıyarak API testi ve hata ayıklamaya yardımcı olur.

---

### 🎨 ColorZilla

:::important[Amaç]
Renkleri seçin ve analiz edin.
:::

**Neden Kullanmalı**: Web siteniz için renk seçmenize ve diğer web sitelerinin renk şemalarını analiz etmenize yardımcı olur. Tutarlı ve çekici bir tasarım oluşturmak için idealdir.

---

### 🧪 Selenyum IDE

:::important[Amaç]
Web testini otomatikleştirin.
:::

**Neden Kullanmalı**: Tarayıcıda otomatik testleri kaydedin ve oynatın. Web sitenizin doğru şekilde çalıştığından emin olmanıza yardımcı olur ve manuel testlerde zaman tasarrufu sağlar.

---

### 🛠️ Web Geliştiricisi

:::important[Amaç]
Web sitelerini yönetin ve inceleyin.
:::

**Neden Kullanmalı**: JavaScript'i devre dışı bırakma, çerezleri inceleme, kaynak kodunu görüntüleme, bağlantıları doğrulama ve daha fazlasına yönelik, tümü web sitelerinde hata ayıklamak ve geliştirmek için önemli olan özellikler sağlar.

---

### 🔤 WhatFont

:::important[Amaç]
Bir web sayfasındaki yazı tiplerini tanımlayın.
:::

**Neden Kullanmalı**: Web sitelerinde kullanılan yazı tiplerini hızlı bir şekilde tanımlar ve yazı tipi özelliklerini görüntüler. Kendi projeleriniz için doğru yazı tiplerini seçip uygulamak için kullanışlıdır.

---

### 🌍 WhatRuns

:::important[Amaç]
Web sitelerinin kullandığı teknolojileri keşfedin.
:::

**Neden Kullanmalı**: Herhangi bir sitede kullanılan eklentileri, widget'ları, analizleri ve çerçeveleri tanımlayın. Trendleri takip etmenize ve kendi projelerinizi geliştirecek araçları keşfetmenize yardımcı olur.

---

## Ön Uç, Arka Uç ve Tam Yığın Geliştiriciler için Ek Araçlar

---

### 📱 React Geliştirici Araçları

:::important[Amaç]
React uygulamalarında hata ayıklayın.
:::

**Neden Kullanmalı**: React uygulamalarında hata ayıklamak için gereklidir; bileşen hiyerarşilerini, durumunu ve özelliklerini incelemenize olanak tanıyarak kullanıcı arayüzünü optimize etmenize yardımcı olur.

---

### 🖥️ Redux Geliştirici Araçları

:::important[Amaç]
Redux durumu değişikliklerinde hata ayıklama.
:::

**Neden Kullanmalı**: Redux destekli uygulamalardaki eylemleri ve durum değişikliklerini izleyin. React ve Redux ile çalışan full-stack geliştiriciler için idealdir.

---

### 🧑‍💻 Vue.js Geliştirme Araçları

:::important[Amaç]
Vue.js uygulamalarında hata ayıklayın.
:::

**Neden Kullanmalı**: Vue bileşenlerini, Vuex durumunu ve daha fazlasını denetlemek için araçlar sağlayarak onu Vue.js kullanan ön uç geliştiriciler için vazgeçilmez kılar.

---

### 🐍 Postacı

:::important[Amaç]
API geliştirme ve test etme.
:::

**Neden Kullanmalı**: API'lere istek göndermeyi, yanıtları görüntülemeyi ve API uç noktalarını test etmeyi kolaylaştırır. Arka uç geliştiriciler veya REST API'leriyle çalışan herkes için gereklidir.

---

### 🖧 Canlı Sunucu

:::important[Amaç]
Yerel bir geliştirme sunucusu başlatın.
:::

**Neden Kullanmalı**: Canlı yeniden yükleme özelliğine sahip yerel bir sunucuyu anında başlatır; ön uç geliştirme sırasında hızlı geri bildirime ihtiyaç duyan tam yığın geliştiriciler için idealdir.

---

### 🌐 GitHub

:::important[Amaç]
GitHub iş akışını geliştirin.
:::

**Neden Kullanmalı**: Bildirimler, çekme istekleri ve sorunlar gibi özellikleri doğrudan tarayıcıdan sunarak işbirliği yapmayı ve depoları yönetmeyi kolaylaştırır.

---

### ⚙️ Gitpod

:::important[Amaç]
Bulutta bir geliştirme ortamı başlatın.
:::

**Neden Kullanmalı**: Tam yığın geliştiriciler için bulut tabanlı bir IDE sağlayarak herhangi bir cihazdan kolaylıkla kod yazmanıza ve GitHub depolarıyla senkronize etmenize olanak tanır.

---

### 📑 JSON Biçimlendirici

:::important[Amaç]
JSON verilerini biçimlendirin ve güzelleştirin.
:::

**Neden Kullanmalı**: JSON verilerini otomatik olarak biçimlendirerek daha okunabilir ve hata ayıklamayı kolaylaştırır. Özellikle API'lerle çalışan arka uç geliştiriciler için kullanışlıdır.

---

### 🖥️ Insomnia

:::important[Amaç]
API tasarımı ve testi.
:::

**Neden Kullanmalı**: RESTful API'leri ve GraphQL'i test etmek için güçlü bir araç. Hata ayıklamayı, istekleri ve yanıtları destekler ve karmaşık API iş akışları oluşturmaya olanak tanır. API odaklı uygulamalarla çalışan geliştiricilerin sahip olması gereken bir şeydir.

---

### 🌍 GitLab

:::important[Amaç]
Kod depoları üzerinde işbirliği yapın.
:::

**Neden Kullanmalı**: Geliştiricilerin projeleri yönetmesine, sorunları takip etmesine ve ekiplerle işbirliği yapmasına olanak tanıyan, Git tabanlı sürüm kontrolü için popüler bir platform. GitLab hem açık kaynak hem de özel projeler için mükemmeldir.

---

### 🔧 Codeberg

:::important[Amaç]
Git deposu barındırma.
:::

**Neden Kullanmalı**: GitHub'a topluluk odaklı bir alternatif olan Codeberg, Git depolarını barındırmak için daha açık kaynak odaklı bir platform sunar. Ana hizmetlere alternatif isteyen geliştiriciler için mükemmel bir seçimdir.

---
