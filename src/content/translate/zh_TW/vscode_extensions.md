## 總覽

探索一系列精選的 VS Code 擴充功能，這些工具旨在提升各個開發領域的生產力並強化技術能力。為方便導覽，這些擴充功能已按類別分組。

---

## 乾淨程式碼與重構 (Clean Code & Refactoring)

:::tip[使用這些擴充功能維護乾淨且易於維護的程式碼，提供重構、度量及改善註解的工具。]
:::

- **Code Metrics**: 直接在狀態列顯示程式碼複雜度，協助進行維護性分析。
- **Abracadabra, Refactor This**: 自動執行重構任務，如在 JavaScript/TypeScript 中重新命名符號或提取變數。
- **Hocus Pocus**: 簡化從使用處建立函式、變數、類別或 switch case 的流程。
- **SonarLint**: 以可執行建議標示程式碼中的錯誤與安全問題。
- **ESLint**: 整合 ESLint 以識別並修復 JavaScript/TypeScript 程式碼中的問題。
- **Better Comments**: 透過色彩標記功能提升註解的可讀性。
- **Document This**: 自動為函式與類別產生 JSDoc 註解。
- **Code Spell Checker**: 偵測程式碼中的拼字錯誤，並提供修正建議。
- **Visual Studio IntelliCode**: 由 AI 驅動的 IntelliSense，提供情境感知建議。
- **TODO Highlight**: 追蹤 TODO 事項與註解，確保不會遺漏任何項目。

## 程式碼片段 (Code Snippets)

:::important
[預先定義的程式碼片段可節省時間並減少工作流程中的重複性任務。]
:::

- **JavaScript (ES6) Code Snippets**: 用於現代 JavaScript/TypeScript 開發的片段。
- **ES7 React/Redux/GraphQL/React-Native Snippets**: 提升 React 與 Redux 生態系統中的生產力。
- **Vue 2/3 Snippets**: 提供 Vue.js 的實用片段。
- **Angular 10 Snippets**: 針對組件、服務等的核心 Angular 片段。
- **Reactjs Code Snippets**: 專為 ES6/ES7 語法的 React 開發而設計。
- **Bootstrap & Font Awesome Snippets**: 用於快速使用 Bootstrap 與 Font Awesome 樣式的片段。
- **Angular Jasmine Unit Testing Snippets**: 使用 Jasmine 加速 Angular 單元測試。
- **Node Snippets**: 用於高效 Node.js 開發的片段。

## 網路工程 (Network Engineering)

:::important
[為您配備專為網路工程任務量身打造的工具。]
:::

- **REST Client**
  允許您直接從編輯器發送 HTTP 請求並檢視回應。適合在不離開 VS Code 的情況下測試 API。
- **Wireshark Syntax Highlighting**
  為 Wireshark 擷取檔案提供語法高亮，使其更易於分析網路流量。
- **Nmap Language Support**
  新增 Nmap 指令稿的語法高亮，協助分析網路弱點與設定。
- **cURL Converter**
  將 cURL 指令轉換為 HTTP 請求，更輕鬆地直接在 VS Code 內測試 API。
- **Telnet Client**
  提供基於終端的 Telnet 用戶端，用於網路疑難排解與遠端伺服器通訊。
- **SSH FS**
  透過 SSH 掛載遠端伺服器，並直接從 VS Code 存取其檔案系統，方便管理。

## Vim 使用者 (Vim Users)

:::tip[透過這些擴充功能讓您的 VS Code 體驗更像 Vim。]
:::

- **Vim**
  將 Vim 快速鍵與功能帶入 VS Code，讓您在編輯器內使用熟悉的 Vim 環境。
- **VsCodium**
  提供受 Vim 啟發的擴充功能，帶來極簡、高度可自訂且可擴充的環境。
- **EasyMotion for VSCode**
  標示目標位置以實現檔案的快速導覽，模擬 Vim 的 EasyMotion 外掛程式。
- **Vimium for VS Code**
  允許您使用鍵盤捷徑導覽 VS Code，就像瀏覽器的 Vimium 一樣。
- **Vim NERDTree**
  將強大的 NERDTree 檔案瀏覽器帶入 VS Code，允許使用 Vim 快速鍵進行檔案導覽與管理。

## 底層程式設計與嵌入式開發 (Low-Level Programming & Embedded Development)

:::note[用於處理組合語言、C 語言與嵌入式系統的擴充功能。]
:::

- **C/C++ Extension Pack**
  一套完整的 C 與 C++ 開發工具，包含 IntelliSense、除錯與程式碼導覽功能。
- **PlatformIO IDE**
  完整的嵌入式系統開發環境，支援廣泛的微控制器與開發板。
- **ARM Assembly Syntax Highlighting**
  為 ARM 組合語言程式碼新增語法高亮，協助視覺化並更輕鬆地理解底層程式碼。
- **x86 and x86_64 Assembly**
  為 x86 組合語言提供語法支援，適合底層程式設計師與逆向工程師。
- **Hex Editor**
  允許您檢視與編輯原始二進位資料，對於處理韌體與底層檔案結構極具價值。
- **AVR Support**
  為 AVR 微控制器程式設計提供支援，包括語法高亮、程式碼片段等。
- **Rust Analyzer**
  Rust 程式語言擴充功能，提供程式碼補全、內嵌文件與重構工具。
- **MicroPython**
  新增 MicroPython 開發支援，包括直接從 VS Code 進行檔案管理與程式碼上傳至裝置。

## 網路安全與逆向工程 (Cybersecurity & Reverse Engineering)

:::warning[分析程式碼弱點、執行靜態分析並探索逆向工程工具。]
:::

- **Binwalk**
  用於分析韌體映像檔的工具，允許您提取與逆向工程嵌入式程式碼與檔案。
- **Ghidra Extension**
  將 Ghidra 逆向工程框架整合至 VS Code，使您能分析二進位檔與反編譯程式碼。
- **YARA Rules**
  為惡意軟體分析與使用 YARA 進行偵測提供語法高亮與基於規則的檔案掃描。
- **Regex Previewer**
  即時視覺化正規表示式及其符合結果，對逆向工程與程式碼分析相當有用。
- **OWASP Dependency Check**
  根據 OWASP 弱點資料庫，掃描您的專案以尋找已知相依套件弱點。
- **Exploit Database Search**
  允許您直接從 VS Code 快速搜尋 Exploit Database 中的已知漏洞。
- **Burp Suite Integration**
  整合 Burp Suite 以強化網頁弱點掃描，提供更流暢的測試工作流程。
- **Metasploit Helper**
  與 Metasploit 互動的輔助整合工具，提升滲透測試效率。

## 效能與分析 (Performance & Profiling)

:::warning[使用這些工具分析並最佳化程式碼效能。]
:::

- **Flamegraph Profiler**
  視覺化 CPU 與記憶體消耗，協助您識別應用程式中的效能瓶頸。
- **Heap Snapshot Viewer**
  透過視覺化 Heap 快照，協助分析記憶體使用情況並尋找記憶體洩漏。
- **Python Profiling**
  提供 Python 的效能分析工具，協助您識別執行緩慢的程式碼並最佳化演算法。
- **Chrome DevTools Integration**
  允許您直接從 VS Code 使用 Chrome DevTools，為網頁開發提供強大的效能分析與除錯工具。
- **Node.js Profiling**
  提供 Node.js 應用程式的效能分析工具，協助您最佳化伺服器端效能。
- **Go Profiler**
  透過視覺化 CPU 使用率、記憶體分配與 goroutines，協助識別 Go 應用程式中的效能問題。
