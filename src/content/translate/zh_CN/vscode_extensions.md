## 概览

探索精选的 VS Code 扩展集合，旨在跨多个开发领域提高生产力并增强技能。这些扩展按类别分组，以便于导航。

---

## 清洁代码与重构

:::tip
使用这些扩展保持代码整洁且易于维护，它们提供了重构、度量和改进注释的工具。
:::

- **Code Metrics**: 直接在状态栏显示代码复杂度，辅助可维护性分析。
- **Abracadabra, Refactor This**: 在 JavaScript/TypeScript 中自动执行重构任务，如重命名符号或提取变量。
- **Hocus Pocus**: 简化从使用处创建函数、变量、类或 switch case 的过程。
- **SonarLint**: 高亮显示代码中的错误和安全问题，并提供可操作的建议。
- **ESLint**: 集成 ESLint，用于识别和修复 JavaScript/TypeScript 代码中的问题。
- **Better Comments**: 通过颜色编码的标签增强注释的可读性。
- **Document This**: 自动为函数和类生成 JSDoc 注释。
- **Code Spell Checker**: 检测代码中的拼写错误，并提供修正建议。
- **Visual Studio IntelliCode**: 基于 AI 的 IntelliSense，提供上下文感知的建议。
- **TODO Highlight**: 跟踪 TODO 和注释，确保不会遗漏任何事项。

## 代码片段

:::important
预定义的代码片段（Snippets）可以节省时间并减少工作流中的重复性任务。
:::

- **JavaScript (ES6) Code Snippets**: 用于现代 JavaScript/TypeScript 开发的代码片段。
- **ES7 React/Redux/GraphQL/React-Native Snippets**: 增强 React 和 Redux 生态系统中的生产力。
- **Vue 2/3 Snippets**: 提供便捷的 Vue.js 代码片段。
- **Angular 10 Snippets**: 用于组件、服务等的必备 Angular 代码片段。
- **Reactjs Code Snippets**: 专为 ES6/ES7 语法的 React 开发而设计。
- **Bootstrap & Font Awesome Snippets**: 用于 Bootstrap 和 Font Awesome 快速样式的代码片段。
- **Angular Jasmine Unit Testing Snippets**: 使用 Jasmine 加速 Angular 单元测试。
- **Node Snippets**: 用于高效 Node.js 开发的代码片段。

## 网络工程

:::important
配备专为网络工程任务定制的工具。
:::

- **REST Client**
  允许你直接从编辑器发送 HTTP 请求并查看响应。非常适合在不离开 VS Code 的情况下测试 API。
- **Wireshark Syntax Highlighting**
  为 Wireshark 捕获文件提供语法高亮，使分析网络流量变得更容易。
- **Nmap Language Support**
  为 Nmap 脚本添加语法高亮，帮助你分析网络漏洞和配置。
- **cURL Converter**
  将 cURL 命令转换为 HTTP 请求，从而更轻松地在 VS Code 内直接测试 API。
- **Telnet Client**
  提供基于终端的 Telnet 客户端，用于网络故障排除和远程服务器通信。
- **SSH FS**
  通过 SSH 挂载远程服务器，并直接在 VS Code 中访问其文件系统，以便于管理。

## Vim 用户

:::tip
使用这些扩展让你的 VS Code 体验更像 Vim。
:::

- **Vim**
  将 Vim 的键位绑定和功能引入 VS Code，让你在编辑器中可以使用熟悉的 Vim 环境。
- **VsCodium**
  提供受 Vim 启发的功能，构建一个极简、高度可定制且可扩展的环境。
- **EasyMotion for VSCode**
  通过高亮目标位置实现文件的快速导航，模拟 Vim 的 EasyMotion 插件。
- **Vimium for VS Code**
  允许你像浏览器中的 Vimium 一样，通过键盘快捷键导航 VS Code。
- **Vim NERDTree**
  将强大的 NERDTree 文件资源管理器引入 VS Code，允许使用 Vim 键位绑定进行文件导航和管理。

## 底层编程与嵌入式开发

:::note
用于汇编、C 语言和嵌入式系统开发的扩展。
:::

- **C/C++ Extension Pack**
  一套用于 C 和 C++ 开发的综合工具，包括 IntelliSense、调试和代码导航。
- **PlatformIO IDE**
  一个完整的嵌入式系统开发环境，支持广泛的微控制器和开发板。
- **ARM Assembly Syntax Highlighting**
  为 ARM 汇编代码添加语法高亮，帮助更直观地理解底层代码。
- **x86 and x86_64 Assembly**
  为 x86 汇编语言提供语法支持，非常适合底层程序员和逆向工程师。
- **Hex Editor**
  允许你查看和编辑原始二进制数据，对于处理固件和底层文件结构极具价值。
- **AVR Support**
  支持 AVR 微控制器编程，包括语法高亮、代码片段等。
- **Rust Analyzer**
  Rust 编程扩展，提供代码补全、内联文档和重构工具支持。
- **MicroPython**
  添加对 MicroPython 开发的支持，包括直接从 VS Code 进行文件管理和设备代码上传。

## 网络安全与逆向工程

:::warning
分析代码漏洞、执行静态分析并探索逆向工程工具。
:::

- **Binwalk**
  用于分析固件映像的工具，允许你提取和逆向嵌入式代码及文件。
- **Ghidra Extension**
  将 Ghidra 逆向工程框架集成到 VS Code 中，支持对二进制文件和反汇编代码进行分析。
- **YARA Rules**
  提供语法高亮和基于规则的文件扫描，用于利用 YARA 进行恶意软件分析和检测。
- **Regex Previewer**
  实时可视化正则表达式及其匹配结果，有助于逆向工程和代码分析。
- **OWASP Dependency Check**
  根据 OWASP 漏洞数据库扫描你的项目，查找依赖项中的已知漏洞。
- **Exploit Database Search**
  允许你直接在 VS Code 中快速搜索 Exploit 数据库中的已知漏洞利用。
- **Burp Suite Integration**
  集成 Burp Suite 以增强 Web 漏洞扫描功能，提供更流畅的测试工作流。
- **Metasploit Helper**
  一个用于与 Metasploit 交互的实用集成，使渗透测试更高效。

## 性能与性能分析

:::warning
使用这些工具分析和优化代码性能。
:::

- **Flamegraph Profiler**
  可视化 CPU 和内存消耗，帮助你识别应用程序中的性能瓶颈。
- **Heap Snapshot Viewer**
  通过可视化堆快照，帮助分析内存使用情况并查找内存泄漏。
- **Python Profiling**
  提供 Python 性能分析工具，帮助你识别缓慢代码并优化算法。
- **Chrome DevTools Integration**
  允许直接从 VS Code 使用 Chrome DevTools，为 Web 开发提供强大的分析和调试功能。
- **Node.js Profiling**
  为 Node.js 应用程序提供分析工具，帮助你优化服务器端性能。
- **Go Profiler**
  通过可视化 CPU 使用率、内存分配和 goroutine，帮助识别 Go 应用程序中的性能问题。
